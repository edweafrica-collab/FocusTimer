---
description: Setup a playground for the new Operator Dashboard experiment
---

This workflow sets up a simplified environment to test the Operator Dashboard UI.

1.  **Create Experiment Directory**
    Create a folder named `dashboard_experiment` in the project root to isolate the new code.
    Command: `mkdir dashboard_experiment` (PowerShell)

2.  **Create HTML File**
    Create `dashboard_experiment/index.html` and paste the provided Dashboard HTML code into it.

3.  **Launch**
    Open `dashboard_experiment/index.html` in a web browser to view the result.
