## Usage Guidelines

### Primary Blue (`#1b2554`)
- Primary buttons and CTAs
- Brand accents
- Interactive elements
- Links and highlights

### Crimson Red (`#d91c26`)
- Warnings and critical alerts
- Error states
- Urgent notifications
- Destructive actions

### White (`#FFFFFF`)
- Primary text
- Timer displays
- UI elements on dark backgrounds
- High contrast elements
